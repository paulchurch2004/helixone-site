Installation sur Mac

3 étapes simples pour lancer HelixOne

1

Téléchargez le .dmg

Cliquez sur "Télécharger pour Mac" et attendez la fin du téléchargement.

2

Glissez dans Applications

Ouvrez le fichier .dmg et glissez HelixOne dans le dossier Applications.

3

Ouvrez l'application

Faites clic droit sur HelixOne > Ouvrir, puis confirmez. Cette étape n'est nécessaire qu'au premier lancement.

macOS affiche un avertissement au premier lancement car l'app n'est pas encore sur l'App Store. Clic droit > Ouvrir permet de contourner ce message en toute sécurité.
